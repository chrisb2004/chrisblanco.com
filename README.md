# chrisblanco.com

This is the source code for my personal portfolio website, where I showcase my projects, skills, and experience as a developer.  
You can visit the live site here: **https://chrisblanco.com**

## Features
- Responsive design for desktop and mobile  
- Sections for About Me, Projects, Skills, and Contact  
- Clean and modern UI  

## Tech Stack
- **Frontend:** HTML, CSS, JavaScript  
- **Hosting:** GitHub Pages

## Installation & Setup
If you want to run this project locally:

```bash
git clone https://github.com/chrisb2004/chrisblanco.com.git
cd chrisblanco.com
```
